js学习日记 Day01

js是一种弱类型的脚本语言，无需编译。它的引入方式分为两种——外部引入、内嵌
js的命名规则：关键词保留字不能做变量名，命名做好做到见名知意

js的数据类型

number 数字 NAN的数据类型是number isNAN()方法返回的是布尔值
string 字符串
boolean 布尔值
undefined 未定义
null 空值

js中的对象是一个无序数据的集合

js中的数据类型转换

强转

parseInt() 从字符的第一位开始看，是否是数字，如果不是，则直接抛出NAN。如果第一位是数字，则保留
parseFloat() 保留小数点
Number() 要看整体是否全是数字，如果是则相加，如果不是则抛出NAN
String() 数字转为字符串
Boolean() 转为布尔值 非数字0都是true，只有数字0才是false

算术运算符

++ 递增 会隐式转换
-- 递减
%  求模取余
+ 拼接，除非两边全是纯数字，不然要先进行隐式转换

赋值运算符

= 赋值
+= a+=2 等同于 a=a+2
-= 
*=
%=
** 求幂

比较运算符

== 只比较值，不比较类型
=== 全相等，既比较值，也比较类型
> 大于
< 小于
>= 大于等于
<= 小于等于
!= 判断值是否不等
！== 判断值是否不全等

三目运算符（三元运算）

条件？表达式一：表达式二 如果条件为真，则执行表达式一，否则执行表达式二

逻辑运算符

&& 与 并且 二者都为真才为真，一假则假
|| 或 二者都为假才为假，一真则真
! 非 取反
( a,b) 只取后面的

分支选择器

单分支if(){}
双分支if(){} else{}
多分支 if(){} else if(){} else{}
switch(需要判定的){
    case 满足条件:执行;breack
}
